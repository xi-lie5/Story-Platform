API Overview

Welcome to the StoryForge API documentation. This API provides access to all the functionality of the StoryForge platform, allowing developers to build applications that interact with our story creation and sharing system.

Base URL

https://api.storyforge.com/api/v1

Data Format

All responses are returned in JSON format.

Authentication

Most endpoints require authentication using JWT tokens.

Authentication APIs

User Registration

POST

Endpoint

/auth/register

Description

Register a new user account on StoryForge.

Request Body

{
  "username": "string",       // 用户名，必填
  "email": "string",          // 电子邮箱，必填
  "password": "string",       // 密码，必填
  "confirmPassword": "string" // 确认密码，必填
}

Success Response

201 Created 
{
  "success": true,
  "message": "注册成功",
  "data": {
    "userId": "string",
    "username": "string",
    "email": "string",
    "token": "string"         // JWT令牌
  }
}

Error Response

400 Bad Request 
{
  "success": false,
  "message": "注册失败",
  "errors": [
    {
      "field": "email",
      "message": "邮箱格式不正确"
    },
    {
      "field": "password",
      "message": "密码长度至少为8位"
    }
  ]
}

User Login

POST

Endpoint

/auth/login

Description

Authenticate a user and get access token.

Request Body

{
  "email": "string",    // 电子邮箱，必填
  "password": "string"  // 密码，必填
}

Success Response

200 OK 
{
  "success": true,
  "message": "登录成功",
  "data": {
    "userId": "string",
    "username": "string",
    "email": "string",
    "token": "string"         // JWT令牌
  }
}

Error Response

401 Unauthorized 
{
  "success": false,
  "message": "邮箱或密码错误"
}

Refresh Token

POST

Endpoint

/auth/refresh

Description

Refresh the access token using a refresh token.

Request Body

{
  "refreshToken": "string"  // 刷新令牌，必填
}

Success Response

200 OK 
{
  "success": true,
  "data": {
    "token": "string",        // 新的访问令牌
    "refreshToken": "string"  // 新的刷新令牌
  }
}

Stories APIs

Get Stories

GET

Endpoint

/stories

Description

Get a list of stories with pagination and filtering.

Query Parameters

Parameter Type Required Description
page number No Page number (default: 1)
limit number No Items per page (default: 10)
category string No Category ID
search string No Search keyword
sort string No Sort by: 'popular', 'latest', 'trending'

Success Response

200 OK 
{
  "success": true,
  "data": {
    "stories": [
      {
        "id": "string",
        "title": "string",
        "author": {
          "id": "string",
          "username": "string"
        },
        "category": {
          "id": "string",
          "name": "string"
        },
        "coverImage": "string",  // 封面图片URL
        "description": "string", // 故事简介
        "rating": number,        // 评分
        "views": number,         // 浏览量
        "createdAt": "string"    // 创建时间
      }
    ],
    "pagination": {
      "total": number,          // 总数量
      "page": number,           // 当前页码
      "limit": number,          // 每页数量
      "pages": number           // 总页数
    }
  }
}

Get Story Detail

GET

Endpoint

/stories/{storyId}

Description

Get detailed information about a specific story.

Path Parameters

Parameter Type Required Description
storyId string Yes The ID of the story

Success Response

200 OK 
{
  "success": true,
  "data": {
    "id": "string",
    "title": "string",
    "author": {
      "id": "string",
      "username": "string",
      "avatar": "string"       // 作者头像URL
    },
    "category": {
      "id": "string",
      "name": "string"
    },
    "coverImage": "string",    // 封面图片URL
    "description": "string",   // 故事简介
    "content": [               // 故事内容段落
      {
        "id": "string",
        "type": "text",        // 内容类型: text, choice
        "text": "string",      // 文本内容
        "choices": [            // 选择项，仅当type为choice时存在
          {
            "id": "string",
            "text": "string",  // 选项文本
            "nextSectionId": "string" // 下一段落ID
          }
        ]
      }
    ],
    "rating": number,          // 评分
    "views": number,           // 浏览量
    "createdAt": "string",    // 创建时间
    "updatedAt": "string"     // 更新时间
  }
}

Error Response

404 Not Found 
{
  "success": false,
  "message": "故事不存在"
}

Create Story

POST

Endpoint

/stories

Description

Create a new story.

Authentication

Required (Bearer Token)

Request Body

{
  "title": "string",           // 故事标题，必填
  "categoryId": "string",     // 分类ID，必填
  "coverImage": "string",     // 封面图片URL，可选
  "description": "string",    // 故事简介，必填
  "content": [                 // 故事内容段落，必填
    {
      "type": "text",          // 内容类型: text, choice
      "text": "string",        // 文本内容，必填
      "choices": [              // 选择项，仅当type为choice时存在
        {
          "text": "string",    // 选项文本，必填
          "nextSectionId": "string" // 下一段落ID，必填
        }
      ]
    }
  ]
}

Success Response

201 Created 
{
  "success": true,
  "message": "故事创建成功",
  "data": {
    "id": "string",           // 新创建故事的ID
    "title": "string"
  }
}

Update Story

PUT

Endpoint

/stories/{storyId}

Description

Update an existing story.

Authentication

Required (Bearer Token)

Path Parameters

Parameter Type Required Description
storyId string Yes The ID of the story

Request Body

{
  "title": "string",           // 故事标题，可选
  "categoryId": "string",     // 分类ID，可选
  "coverImage": "string",     // 封面图片URL，可选
  "description": "string",    // 故事简介，可选
  "content": [                 // 故事内容段落，可选
    {
      "id": "string",         // 段落ID，更新时必填
      "type": "text",         // 内容类型: text, choice
      "text": "string",       // 文本内容
      "choices": [             // 选择项，仅当type为choice时存在
        {
          "id": "string",     // 选项ID，更新时必填
          "text": "string",   // 选项文本
          "nextSectionId": "string" // 下一段落ID
        }
      ]
    }
  ]
}

Success Response

200 OK 
{
  "success": true,
  "message": "故事更新成功"
}

Error Response

403 Forbidden 
{
  "success": false,
  "message": "没有权限更新此故事"
}

Add Story Section

POST

Endpoint

/stories/{storyId}/sections

Description

Add a new section to a story.

Authentication

Required (Bearer Token)

Path Parameters

Parameter Type Required Description
storyId string Yes The ID of the story

Request Body

{
  "type": "text",              // 内容类型: text, choice
  "text": "string",            // 文本内容，必填
  "choices": [                  // 选择项，仅当type为choice时存在
    {
      "text": "string",        // 选项文本，必填
      "nextSectionId": "string" // 下一段落ID，必填
    }
  ],
  "isEnd": boolean              // 是否为故事结尾，可选，默认false
}

Success Response

201 Created 
{
  "success": true,
  "data": {
    "id": "string",           // 新添加段落的ID
    "message": "段落添加成功"
  }
