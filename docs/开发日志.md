# 开发日志

## 2026-05-18
- 启用 Helmet，补齐基础安全响应头。
- 修复刷新令牌校验与 MySQL 模型兼容性（移除 `.select`、使用 `id` 与本地 refreshToken 字段）。
- 前端 API 基础地址支持运行时注入（`window.__API_BASE__` / `window.API_BASE_URL`）。
- `authGuard` 调试日志仅在非生产环境输出。
- **新增：阅读进度持久化系统** — 基于 localStorage 自动保存用户在故事阅读器中的阅读节点位置，支持跨会话恢复。
- **新增：继续阅读快捷入口** — 首页和浏览页展示最近阅读的 4 个故事卡片，含进度条和直接跳转链接，无记录时自动隐藏。
- **新增：阅读历史管理** — 支持查看最近 10 条阅读记录、单条删除、全部清空，记录含时间戳和节点信息。
- **新增：全局深色/浅色主题切换** — 通过导航栏主题切换按钮一键切换，偏好持久化到 localStorage，覆盖全部 7 个前端页面。
- 新增 `front/assets/js/reading-history.js` 通用模块，封装阅读进度和主题管理。
- 修复 `front/browse.html` 重复引入 `navbar.js` 的问题。
- 修复 `front/story-reader.html` 主题切换按钮默认隐藏的问题。
- **V10 前端视觉重构：温润书房风格** — 全面重新设计前端视觉体系，统一采用「文学杂志 × 旧图书馆」美学方向。
  - 新增 `front/assets/css/design-system.css` 全局设计系统，定义 CSS 变量（暖奶油底、琥珀金强调、墨绿点缀、纸质纹理）。
  - 字体切换为 **Playfair Display**（标题衬线）+ **Lora**（正文衬线），替代原 Inter / 系统字体。
  - 导航栏改为温润木质调毛玻璃（`nav-library`），按钮体系统一为 `.btn` / `.btn-icon` 组件。
  - 首页（`index.html`）完全重写：层叠书本视觉、装饰性引号、纸质纹理卡片、错落入场动画。
  - 阅读器（`story-reader.html`）改为书页体验：暖白书页背景、首字下沉（drop cap）、左侧装饰条、2em 段落缩进。
  - 浏览页（`browse.html`）改为图书馆目录风格：暖白卡片、琥珀色分类标签、装饰性分割线。
  - 登录/注册页改为居中书房卡片布局：纸质纹理、琥珀 / 鼠尾草色调图标。
  - 功能页面（`create.html`、`my_stories.html`、`profile.html`）接入设计系统颜色令牌和字体。
  - 深色模式适配全部页面：深棕底色 + 暖光文字 + 琥珀色强调。
  - `navbar.js` 导航栏注入逻辑更新为 `nav-link` 新样式，主题切换按钮改用 `btn-icon` 组件。
